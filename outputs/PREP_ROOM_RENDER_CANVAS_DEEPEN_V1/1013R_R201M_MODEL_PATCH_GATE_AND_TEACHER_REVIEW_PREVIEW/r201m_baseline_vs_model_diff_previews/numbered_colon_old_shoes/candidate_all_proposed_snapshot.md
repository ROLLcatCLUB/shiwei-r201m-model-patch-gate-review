# 旧鞋 / 足下生辉 candidate all-proposed preview

## 一、本课依据

1. 本课依据上传原稿定位为三年级《足下生辉》旧鞋改造设计课，教学主线是把上一课草图转化为旧鞋改造作品。
2. 本课学习重心不是鞋的线描观察，而是围绕旧鞋材料、结构、设计意图、设计故事和发布会准备，完成一次设计应用表达。

## 二、学情分析

1. 三年级学生对旧物改造有兴趣，多数已带上一课设计草图进入课堂，容易把旧鞋改造做成随意贴饰，忽略材料是否回应原设计意图。
2. 本课需要持续追问'你想改造成什么、为什么这样选材料、哪里体现设计意图'，把制作过程转化为可说明的设计证据。

## 三、教学目标

1. 学生能对照上一课设计草图，说出旧鞋要改造的部位和制作路径。
2. 学生能根据设计意图选择旧材料，完成旧鞋作品的主体制作。
3. 学生能用5句话设计故事说明作品对象、解决的问题、材料方法、满意处和改进想法。

## 四、教学重难点

1. 重点：把上一课草图中的设计意图转化为旧鞋作品，落实到材料选择和制作路径上。
2. 难点：让学生在制作中不断回应设计意图，材料方法与设计想法保持一致，而不仅追求外观好看。

## 五、教学准备

1. 学生准备旧鞋、上一课设计草图、可选装饰材料、故事卡或记录单。
2. 教师准备材料连接示例、工具安全提醒、5句话设计故事句式和发布会作品整理要求。

## 六、教学过程

### 1. 回顾草图与任务确认

- 环节目标：学生能对照上一课草图，确认今天旧鞋改造的对象和设计意图。
- 教师组织：组织学生拿出草图，先说准备改造成什么，再圈出今天最要实现的一处设计重点。
- 学生学习：查看草图，说明旧鞋改造方向，并确认一处需要重点完成的设计内容。
- 关键话术：今天不是重新画一双鞋，而是把草图里的想法真正做出来。先说说你想把这只旧鞋改造成什么？
- 核心证据：学生能拿出草图并说明旧鞋改造方向。

### 2. 制作路径讨论

- 环节目标：学生能比较粘贴、包裹、添加、绘制等方法，判断哪一种更适合自己的设计。
- 教师组织：引导学生把材料方法和设计意图配对，讨论不同连接方式的稳定性和表现效果。
- 学生学习：比较制作路径，说明自己选择某种材料或方法的理由。
- 关键话术：如果你想表现这个功能或造型，粘贴、包裹、添加、绘制哪一种更能支持你的想法？为什么？
- 核心证据：学生能比较粘贴、包裹、添加、绘制等制作路径，并说出所选方法如何支持自己的设计意图。

### 3. 旧鞋作品制作

- 环节目标：学生能根据草图完成旧鞋改造，并在制作中检查结构稳定和材料效果。
- 教师组织：巡视旧鞋作品制作，重点关注材料连接、工具安全和作品是否还在回应草图设计意图。
- 学生学习：按照草图进行旧鞋改造，遇到连接或装饰问题时说明困难并调整。
- 关键话术：现在这一步有没有帮助你实现草图里的设计？如果只是贴上去好看，它和你的想法有什么关系？
- 核心证据：作品能看到旧鞋改造痕迹，并有至少一处材料或结构处理回应草图。

### 4. 5句话设计故事

- 环节目标：学生能用五句话说明作品对象、问题、材料方法、满意处和改进点。
- 教师组织：提供五句话结构，让学生把制作结果转化为发布会可以讲清楚的设计故事。
- 学生学习：围绕作品对象、解决的问题、材料方法、满意处和改进处，写出5句话设计故事，并对照作品检查每句是否有证据。
- 关键话术：请按五句话说：它是谁、解决什么问题、用了什么材料方法、你最满意哪里、还想改什么。
- 核心证据：学生完成故事卡，能对应作品中的材料和结构证据。

### 5. 新鞋发布会准备

- 环节目标：学生能整理作品和故事卡，为下一环节展示发布做好准备。
- 教师组织：组织学生检查作品、故事卡和展示顺序，提醒发布会要用作品证据支持介绍。
- 学生学习：整理旧鞋作品与故事卡，准备展示时的介绍顺序。
- 关键话术：新鞋发布会上，请把旧鞋作品和故事卡一起准备好，先讲作品对象，再讲你解决了什么问题，材料方法是怎样的。
- 核心证据：学生能准备好作品和故事卡，并说出展示时先讲哪一点。

## 七、学习单与评价

1. 过程证据：学生能对照草图选择材料和制作方法。
2. 作品证据：旧鞋作品中能看到材料选择、结构处理或装饰表达回应设计意图。
3. 表达证据：5句话设计故事能说明对象、问题、材料方法、满意处和改进点。
