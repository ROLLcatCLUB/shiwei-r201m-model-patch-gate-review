# 穿穿编编 candidate all-proposed preview

## 一、本课依据

1. 本课围绕编织历史、经纬交织原理和纸条编织练习展开，让学生从实物观察进入基本方法体验。
2. 原文提供导入、历史介绍、认识经纬、教师示范、学生练习和小结六个环节，教学主线应从文化认识走向方法操作。

## 二、学情分析

1. 四年级学生能观察编织实物的纹理，也具备剪贴和纸条操作经验，但未必理解经线固定、纬线穿插形成图案的原理。
2. 本课需要把历史介绍控制为短背景，把重点落到经纬关系、压一穿一方法和纸条练习中的手眼协调。

## 三、教学目标

1. 学生能说出编织的1-2个历史应用（如席子、结绳记事），把编织与生活或记录联系起来。
2. 学生能区分经线（竖向固定）和纬线（横向穿过），理解压一穿一形成交织纹样的原理。
3. 学生能在底稿纸上完成一段纸条经纬交织练习，并在小结中说出一个操作要点或仍需改进的位置。

## 四、教学重难点

1. 重点：认识经线与纬线的固定与穿插关系，掌握纸条'压一穿一'的基本编织方法。
2. 难点：在连续编织中保持一上一下交替不乱，并能向同伴指出一处自己调整过的位置。

## 五、教学准备

1. 教师准备：编织实物（竹编/藤编样品），如无实物改用图片；经纬示意图或PPT；编织底稿纸（经线底格，每生1张）。
2. 学生准备：各色纸条（1.5cm宽，预先裁好）、底稿纸、剪刀，按课堂常规使用剪刀，注意工具安全。

## 六、教学过程

### 1. 导入

- 环节目标：学生能从编织实物中发现交错纹理，产生学习编织方法的兴趣。
- 教师组织：出示竹编或藤编实物（无实物则用图片替代），先让学生说说它是什么、用来做什么，再请他们观察表面一条一条材料是怎样穿在一起的。
- 学生学习：观察实物，说出在哪里见过类似编织物，并指出交错纹理。
- 关键话术：它不是画出来的纹样，而是一条一条材料穿出来的。你能找到哪条在上、哪条在下吗？
- 核心证据：学生能指出实物中的交错纹理。

### 2. 历史介绍

- 环节目标：学生能知道编织与生活、记录和装饰有关，理解它不是单纯手工小练习。
- 教师组织：用2-3张图按顺序介绍：七千年前席子→结绳记事→彩陶编织纹样，每个例子只说一两句生活或记录中的作用，不展开成大段讲授。
- 学生学习：听老师介绍后，用一两句话在同桌或小组里说自己最记得的一个编织用途（席子/结绳记事/彩陶纹样中的一个）。
- 关键话术：古人编席子是为了解决生活需要，后来这些交错纹理也变成了装饰。你觉得哪一种最接近今天的纸编？
- 核心证据：学生能说出一个编织的生活用途或文化背景。

### 3. 认识经纬

- 环节目标：学生能区分经线和纬线，理解纬线穿过经线形成交织关系。
- 教师组织：用示意图和纸条演示经线竖向固定、纬线横向穿插，强调上下交替。
- 学生学习：用手势或纸条模拟经纬方向，说出压一穿一的规律。
- 关键话术：竖着不动的是经线，横着来回穿的是纬线。下一格如果在上面，后一格应该在哪里？
- 核心证据：学生能正确指出经线、纬线，并说出上下交替规律。

### 4. 教师示范

- 环节目标：学生能看懂纸条编织的起步、穿插和收边方法。
- 教师组织：分步示范固定经线、穿入纬线、压一穿一和整理边缘，控制速度让学生看清手法。
- 学生学习：跟随示范在草稿纸上试做一到两步，并提出不清楚的位置。
- 关键话术：这一条穿过去时不要急着粘，先检查是不是一上一下交替，再把纸条推整齐。
- 核心证据：学生能完成一段压一穿一试排。

### 5. 学生练习

- 环节目标：学生能独立完成一段纸条编织，并调整纸条松紧和排列。
- 教师组织：在教室中巡视，重点检查：纸条是否一上一下交替、纬线是否横穿到底、经线有没有被带动移位；对连续两次压错的同学做一次小范围重示范。
- 学生学习：完成纸条编织练习，检查是否有连续压错或缝隙过大的问题。
- 关键话术：先别急着贴牢，沿着这一行检查：有没有连续两次都在上面？哪里需要推紧一点？
- 核心证据：作品上能看到经纬上下交替的纹理，学生能口头指出至少一处自己调整过（例如某格重穿、某条纸条推紧）的地方。

### 6. 小结

- 环节目标：学生能用自己的话说出经纬关系和一个编织操作要点。
- 教师组织：组织学生展示练习片段，围绕经线、纬线、压一穿一和整理边缘做小结。
- 学生学习：展示练习，说出自己学会的一个方法和仍需改进的一处。
- 关键话术：今天你不是只完成了一张纸编，而是学会了经线和纬线怎样合作。你最容易出错的是哪一步？
- 核心证据：学生能说出一个经纬关系或操作要点。

## 七、学习单与评价

1. 认知证据：学生能区分经线和纬线。
2. 操作证据：纸条编织练习体现压一穿一的上下交替。
3. 表达证据：学生能说出一个编织操作要点或困难。
